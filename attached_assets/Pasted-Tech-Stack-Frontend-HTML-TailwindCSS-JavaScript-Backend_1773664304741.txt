Tech Stack
Frontend: HTML, TailwindCSS, JavaScript
Backend: Python (Flask or FastAPI)
Map: Leaflet.js with OpenStreetMap tiles
Requirements
1️⃣ Full Screen Responsive Layout
The website must occupy 100% width and 100% height of the screen.
No blank margins.
Map should automatically expand to fill the entire screen.
Use TailwindCSS responsive design.
The layout must adjust to mobile, tablet, and desktop screens.
2️⃣ Google-Maps-Like Map Interface
The map must look and behave like Google Maps UI.
Use Leaflet.js map with OpenStreetMap tiles.
Enable:
Zoom in/out controls
Smooth map panning
Dragging the map
Marker animations
The map must open fully when the website loads.
3️⃣ Real-Time Bus Tracking
A bus marker icon should be visible on the map.
The marker must move slowly like a GPS tracker.
Update the bus location every 2 seconds using simulated coordinates from backend.
Movement should be smooth, not jumping.
The path should show the route line.
