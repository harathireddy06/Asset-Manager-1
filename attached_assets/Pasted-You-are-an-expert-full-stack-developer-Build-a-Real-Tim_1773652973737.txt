You are an expert full-stack developer. Build a Real-Time Public Transport Tracking Web Application for small cities and villages in Telangana.

The system should allow villagers to track buses in real time, view estimated arrival time, and receive alerts and notifications.

The interface must be very simple so even uneducated people can understand it.

4 Real-Time GPS Tracking

Simulate live GPS tracking.

The bus marker should move every 2 seconds.

Example JavaScript:

setInterval(() => {
    fetch('/bus-location')
    .then(res => res.json())
    .then(data => {
        updateBusMarker(data.lat, data.lng);
    });
}, 2000);

Backend should generate new GPS coordinates every 2 seconds.

5 Estimated Time of Arrival (ETA)

After user selects From → To, show:

Bus Number
Distance Remaining
Estimated Arrival Time

Example UI:

Bus Number: TS05-2345
Distance Remaining: 12 km
Estimated Arrival: 18 minutes

ETA formula:

ETA = distance / average bus speed

Use average speed = 40 km/h.
